Research

Ultra-fast Seam Carving for Real Time Object Detection and Removal

IEEE International Conference on Computational Photography(ICCP)(2020) Poster

Nicole Kosoris, Disha Das, Naila Fatima

This paper describes a new method for quickly removing objects from videos in real-time. By enhancing the Seam Carving algorithm with multiprocessing and the "Spiral Model," objects can be removed in just 35 milliseconds with minimal processing delay, ensuring smooth video streaming.

Spider Robot Trajectory Optimization using Factor Graphs

CS6999 Master's Project (2020)

Disha Das, Tarushree Gandhi

In this project, we executed trajectory optimization of a spider robot model using the GTDynamics library, a library that allows implementation of kinodynamic constraints on robot configurations using factor graphs.

Evaluation of 3D Object Detection for Cars by improving existing frameworks

CS7643 Deep Learning final project (2020)

Sanmeshkumar Udhayakumar, Disha Das, Tarushree Gandhi, Leon Carter Price

In our study, we address 3D car object detection using the Kitti dataset. Employing PointRCNN and Frustum ConvNet, we aim to enhance their performance through experiments. Our findings reveal improved results, particularly in the medium and hard 3D Average Precision categories, with PointRCNN excelling in the latter and Frustum ConvNet in the former.

Human Acceptance of Unpredictable Behaviors in Fully Driverless Ride-Hailing Services

CS7633 Human Robot Interaction final project (2020)

Disha Das, Tarushree Gandhi

Research has focused on human trust in Autonomous Vehicles through surveys assessing experience, likeability, and trust. While studies examine factors affecting trust, such as usage frequency and ease of learning, none have compared acceptance of errors between human-driven and autonomous cars. This study aims to fill this gap, revealing that humans are more forgiving of mistakes made by autonomous cars compared to human drivers.

Comparing the performance of Generative Models for Vector Drawing generation

CS7641 Machine Learning final project (2020)

Disha Das, Ashwin Jeyaseelan, Naila Fatima, Tarun Pasumarthi

The aim of the project is to develop machine learning models capable of generating sketches across various classes, exploring variations such as VAE-CNN, DCGAN, and Sketch-pix2seq alongside Sketch-RNN, to assess their recognizability by trained classifiers.

Obstacle Avoidance for Kinematically Redundant Mobile Manipulators

CS8803 Mobile Manipulation final project (2020)

Disha Das, Tarushree Gandhi

This project focuses on using redundant degrees of freedom in a 6-link mobile manipulator to avoid obstacles during task execution. It employs obstacle avoidance algorithms implemented with the GTSAM library to ensure collision-free trajectory planning for the end-effector.

Identifying Misogynistic Rhetoric Within Advice-Seeking Communities

CSE6240 Web Search and Text Mining final project (2020)

Disha Das, Tanvi Bhagwat, Aiswarya Bhagavatula

Reddit facilitates the discovery of niche communities, empowering users to engage anonymously in discussions they may not feel comfortable having publicly. While this anonymity fosters open dialogue, it also enables the spread of harmful ideologies, evident in communities where personal issues are discussed collaboratively. Analyzing Reddit's API data reveals a rise in negative discourse within these safe spaces, potentially deterring user engagement if not addressed.

Enhancing Image Contrast: A Novel Approach Incorporating 2D Histogram Equalization

Bachelor's Thesis (2016)

Disha Das, Chetana Pradhan, Hiten Kumar Behera

This paper introduces a novel contrast enhancement method that improves upon traditional 1D Histogram Equalization (HE) by considering 2D Histogram Equalization (2DHE) to address artifacts. The study compares the effectiveness of existing techniques with 2DHE, showcasing its ability to enhance local contrast and dynamic range for grayscale and color images with minimal complexity.
